üîç SYSTEM ASSUMPTIONS

  • User Registration: Student must register before accessing any content
  • Authentication: JWT-based session management for secure login
  • Free Trial: New students get 2 sample videos, 2 notes, 1 mock test for free
  • Subscription Required: Full content access requires active paid subscription
  • Payment Gateway: Integrated with UPI/Card/NetBanking for online payments
  • Automatic Evaluation: Exams are auto-evaluated and results generated instantly
  • Content Lock: Paid content is locked for trial/expired users
  • Notification System: Automated alerts for exams, payments, subscription expiry
  • Progress Tracking: System maintains performance analytics for each student
  • Data Security: Passwords encrypted, payment data secured via gateway
